แก้ปัญหาภาษาไทยใน End Note X9

  1. ไปที่ Edit > Preferences...
  2. เลือกหัวข้อ Display Fonts ทางซ้าย จากนั้นเลือกแทบ General ทางขวามือ
  3. เลือก Font ที่มีภาษาไทยเช่น Cordia New จากนั้นเลือก Script เป็น Thai
  4. กด OK ก็เรียบร้อย
